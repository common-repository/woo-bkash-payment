== Description ==

This is a plugin to shop online around Bangladesh using bKash payment gateway. If you(Store owner)'re using WooCommerce for your store, you should choose bKash for local payment for your shop

First install woocommerce plugin then install bkash plugin and customize bkash payment options

Full Documentation see more screenshots


== Installation ==

Here is how you should install the plugin

e.g

1. Upload the plugin files to the `/wp-content/plugins/Woo-Bkash-Payment.zip` directory, or install the plugin through the WordPress plugins screen directly.
2. then activate the plugin and enjoy
